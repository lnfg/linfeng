修仙之记录下自己搭建的个人博客

  • 随笔

虽然网上也有很多的教程，但是还是想把自己的记录一下，免得n年之后忘记了。这也是一种学习，一种成长。

  1. 博主系统配置
  2. 必备软件
  3. github账号申请与pages的操作
  4. hexo主题
  5. 域名解析【倘若你已经购买域名】

1.博主系统配置

  • windows 10 系统(x64)

2.必备软件

  • Node.js
  • Git
  • hexo-cli

搭建个人博客开始了，赶紧上车！开车了！

嘟嘟嘟。。。。。。
没时间解释了。。。赶紧上车！

  • 1.先安装 强大的 Node.js先吧！

    下载链接
    Node.js
    最好是下载的是稳定版本，至于安装的步骤就不详解了。相信你是最棒的！博主的版本是v6.10.2

  • 2.安装Git

    下载链接 Git 博主下载的版本是git version 2.12.0.windows.1

    前面两个装好了之后，可以调出cmd，输入命令npm -v ,可以输出的是4.5.0（博主npm的版本）
    哎哟，我去，废话真多！

  • 3.安装hexo-cli【这个是用于生成hexo主题、生成静态页面和部署到github上，一个命令就搞定了】

    hexo官网
    在cmd中执行命令 F:\LnfgBlog>npm install hexo-cli -g

    切换到自己要存放博客文件的目录，比如：F:\LnfgBlog 文件夹下。

    初始化hexo 博客 F:\LnfgBlog>hexo init blog 然后在LnfgBlog目录下得到一个名为blog的文件夹。里面有相关的代码。

    继续执行 F:\LnfgBlog>npm install

    然后执行 F:\LnfgBlog>hexo server

    然后复制 链接 http://localhost:4000/
    到网址上就可以显示一个简单的界面了。

    就是那么简单

3.github账号申请与pages的操作

①

到Github上申请一个账号。
然后我们登陆，可以到这里看官方的教程。

  • 到这儿 创建仓库 亦即是 你的 博客项目名称。

  • 注意：博客项目名称必须是照

    {username}.github.ui

-命名。创建好之后呢，就可以使用git 操作了。

②

  • 克隆博客项目到本地
    使用 git 操作，在你想要的目录下 打开 git bash，然后

    ~ $ git clone https://github.com/username/username.github.io

  • 随便添加点内容

    `~ $ cd username.github.io

    ~$ echo "Hello World" > index.html

  • 添加，提交和上传到github上

    ~ $ git add --all

    ~ $ git commit -m "第一次提交项目"

    ~ $ git push -u origin master

  • 然后看看github 上的 博客是不是已经添加了新的文件.

  • 然后直接在网址栏输入 username.github.io 会出现你自己的个人博客
